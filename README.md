## üåç **Why LoopFiX**

Traditional savings systems are slow, centralized, and offer minimal returns. DeFi protocols promise higher yields ‚Äî but they're complex, fragmented, and risky for everyday users.

**LoopFiX bridges this gap** by:
- Turning **familiar savings habits** into **yield-generating strategies**
- Using **AI-powered advisors** to guide users safely
- Leveraging **Bitcoin's security** for peace of mind
- Enabling **group-based social finance** to boost commitment and trust

## üß≠ **User Flow**

1. **Connect Wallet** ‚Üí User authenticates with Leather/Hiro  
2. **Create or Join Pool** ‚Üí Individual savings goal or group vault  
3. **Deposit STX** ‚Üí Funds locked in Clarity smart contracts  
4. **Track Progress / Earn Yield** ‚Üí Real-time updates via backend  
5. **Withdraw or Reinvest** ‚Üí Once target or duration is reached  
6. **Optional AI Advice** ‚Üí For smarter reinvestment strategies

### **Installation**

```bash
# Clone repository
git clone https://github.com/yourusername/loopfi.git
cd loopfi

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Install contract dependencies
cd ../contracts
```

**Frontend (.env)**
```env
VITE_STACKS_NETWORK=testnet
VITE_STACKS_API_URL=https://stacks-node-api.testnet.stacks.co
```
